# ElevateCV
**Transform Your Resume Into A Stunning Portfolio**

## Features
- Upload PDF/TXT resume
- Template thumbnails (Modern / Executive / Academic)
- Demo resume button
- Progress stepper (Upload → Parse → Generate → Ready)
- Exports: HTML + PDF
- Better PDF extraction using PDF.js (client-side)
- Server fallback parsing for PDFs
- Rate limiting (basic)

## Deploy (Vercel)
1. Push this repo to GitHub
2. Import into Vercel
3. Add Environment Variable:
   - `OPENAI_API_KEY` = your key
4. Deploy

## Local test (optional)
This is a static + serverless project (Vercel-style). Best tested by deploying to Vercel.
